## 1. 支出-總金額

#### 程式名稱: hw09_1.py
``` python
(1) 建立一個[支出]類別
    (1-1) 類別名稱: Expense
    (1-2) 建構元: 傳入[日期](expDate), [金額](*price) 等2個參數, 其中金額可存入多個參數值.
                  將參數存入實體中, 成為實體的成員.
    (1-3) [計算總金額]方法, total(), 
          其值 = 所有支出金額的累加.          

(2) 產生實體
(3) 顯示資料
```

#### 測試
``` python
'2020-3-1', 80, 100, 70, 65, 140 
'2020-3-2', 180, 75, 40
'2020-3-3', 85, 40, 95, 105, 55, 90, 210, 450, 300
'2020-3-4', 30, 45, 35, 70, 70
'2020-3-5', 120, 80, 45, 95, 65, 40
'2020-3-6', 180, 210    
```

#### 輸出
``` python
日期:2020-3-1, 總支出:455元
日期:2020-3-2, 總支出:295元
日期:2020-3-3, 總支出:1,430元
日期:2020-3-4, 總支出:250元
日期:2020-3-5, 總支出:445元
日期:2020-3-6, 總支出:390元
```


## 2. 支出-平均金額

#### 程式名稱: hw09_2.py
``` python
(1) 建立一個[支出]類別
    (1-1) 類別名稱: Expense
    (1-2) 建構元: 傳入[日期](expDate), [金額](*price) 等2個參數, 其中金額可存入多個參數值.
                  將參數存入實體中, 成為實體的成員.
    (1-3) [計算總金額]方法, total(), 
          其值 = 所有支出金額的累加.  
    (1-4) [計算平均支出金額]方法, average(), 
          其值 = 所有支出金額的累加 / 支出次數.          

(2) 產生實體
(3) 顯示資料, [平均支出金額]計算至整數, 餘四捨五入. 
```

#### 測試
``` python
'2020-3-1', 80, 100, 70, 65, 140 
'2020-3-2', 180, 75, 40
'2020-3-3', 85, 40, 95, 105, 55, 90, 210, 450, 300
'2020-3-4', 30, 45, 35, 70, 70
'2020-3-5', 120, 80, 45, 95, 65, 40
'2020-3-6', 180, 210    
```

#### 輸出
``` python
日期:2020-3-1, 總支出:455元, 支出項目:5, 平均支出:91元
日期:2020-3-2, 總支出:295元, 支出項目:3, 平均支出:98元
日期:2020-3-3, 總支出:1,430元, 支出項目:9, 平均支出:159元
日期:2020-3-4, 總支出:250元, 支出項目:5, 平均支出:50元
日期:2020-3-5, 總支出:445元, 支出項目:6, 平均支出:74元
日期:2020-3-6, 總支出:390元, 支出項目:2, 平均支出:195元
```
