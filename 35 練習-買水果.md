## 1. '蘋果', '香蕉', '橘子', '鳳梨', '芭樂'

#### 程式名稱: hw35_1.py
``` python
問題: 小明想在'蘋果', '香蕉', '橘子', '鳳梨', '芭樂'等5種水果中買不重覆的3樣, 預算是150~200元之間.
      如果:蘋果60元, 香蕉15元, 橘子25元, 鳳梨100元, 芭樂35元.
      在小明的預算範圍中, 能買夠怎麼買?
``

#### 參考答案
``` python
('蘋果', '香蕉', '鳳梨'), 共175元
('蘋果', '橘子', '鳳梨'), 共185元
('蘋果', '鳳梨', '芭樂'), 共195元
('香蕉', '鳳梨', '芭樂'), 共150元
('橘子', '鳳梨', '芭樂'), 共160元
5
```
