## 1. 一個員工類別

#### 1-1 學習重點
``` python
(1) 儲存架構
      |__ p01.py

(2) 這個員工類別並沒有使用繼承.
```

#### 1-2 程式範例(p01.py)

``` python
#---------------------------
# 員工[類別]
#---------------------------
class Employee():
    # 建構元
    def __init__(self, no, name, rank='C'):
        self.no = no
        self.name = name
        self.rank = rank

    # 計算薪水
    def salary(self):
        if self.rank=='A':
            return 50000
        elif self.rank=='B':
            return 40000
        else:
            return 30000

#===========================
# 主程式
#===========================
# 建立物件
e = Employee('001', '王小明', 'B')
print('編號:{}'.format(e.no))
print('姓名:{}'.format(e.name))
print('職等:{}'.format(e.rank))
print('薪水:{:,}元'.format(e.salary()))
```

#### 1-3 執行結果
``` python
>> 編號:001
>> 姓名:王小明
>> 職等:B
>> 薪水:40,000元
```
