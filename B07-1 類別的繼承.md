# B07-1 類別的繼承

## 1. 一個員工類別

#### 1-1 學習重點
``` python
(1) 儲存架構
      |__ p01.py

(2) 這個員工類別並沒有使用繼承.
```

#### 1-2 程式範例(p01.py)

``` python
#---------------------------
# 員工[類別]
#---------------------------
class Employee():
    # 建構元
    def __init__(self, no, name, rank='C'):
        self.no = no
        self.name = name
        self.rank = rank

    # 計算薪水
    def salary(self):
        if self.rank=='A':
            return 50000
        elif self.rank=='B':
            return 40000
        else:
            return 30000

#===========================
# 主程式
#===========================
# 建立物件
e = Employee('001', '王小明', 'B')
print('編號:{}'.format(e.no))
print('姓名:{}'.format(e.name))
print('職等:{}'.format(e.rank))
print('薪水:{:,}元'.format(e.salary()))
```

#### 1-3 執行結果
``` python
>> 編號:001
>> 姓名:王小明
>> 職等:B
>> 薪水:40,000元
```

## 2. 一個員工類別

#### 2-1 學習重點
``` python
(1) 儲存架構
      |__ p01.py

(2) [海外員工]繼承了[員工]類別, 寫法:
    class OverseasEmployee(Employee):

(3) 在[海外員工]呼叫父親的建構元, 寫法:
    super().__init__(no, name, rank)
    
(4) 在[海外員工]呼叫父親的薪水函式, 寫法:
    super().salary()   
```

#### 2-2 程式範例(p01.py)

``` python
#---------------------------
# 員工[父類別]
#---------------------------
class Employee():
    # 建構元
    def __init__(self, no, name, rank='C'):
        self.no = no
        self.name = name
        self.rank = rank

    # 薪水
    def salary(self):
        if self.rank=='A':
            return 50000
        elif self.rank=='B':
            return 40000
        else:
            return 30000

#---------------------------
# 海外員工[子類別]
#---------------------------
class OverseasEmployee(Employee):
    # 建構元
    def __init__(self, no, name, rank='C', country='US'):
        super().__init__(no, name, rank)
        self.country = country

    # 薪水
    def salary(self):
        if self.country=='US':
            return super().salary()+25000
        else:
            return super().salary()+15000

#===========================
# 主程式
#===========================
# 建立[員工]物件
e = Employee('001', '王小明', 'B')
print('編號:{}'.format(e.no))
print('姓名:{}'.format(e.name))
print('職等:{}'.format(e.rank))
print('薪水:{:,}元'.format(e.salary()))
print()

# 建立[海外員工]物件
oe = OverseasEmployee('002', '陳小華', 'A', 'US')
print('編號:{}'.format(oe.no))
print('姓名:{}'.format(oe.name))
print('職等:{}'.format(oe.rank))
print('外派國家:{}'.format(oe.rank))
print('薪水:{:,}元'.format(oe.salary()))
```

#### 2-3 執行結果
``` python
>> 編號:001
>> 姓名:王小明
>> 職等:B
>> 薪水:40,000元
>> 
>> 編號:002
>> 姓名:陳小華
>> 職等:A
>> 外派國家:A
>> 薪水:75,000元
```
