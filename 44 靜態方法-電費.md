## 1. 電費-單一費率


#### 儲存結構
``` python
  |
  |__<ntub>
  |     |__ m01.py
  |
  |__ p01.py   
```


#### 程式名稱: m01.py (自行完成)
``` python
(1) 類別名稱: Fee
(2) 靜態方法名稱: single(degree)
    [2-1] 傳入參數: 用電度數(int) 
    [2-2] 回傳值: 電費(int)
    [2-3] 說明: 
          註:
          ...................................
                               每度（元）   
          ...................................
          330度(含)以下         3.02               
          331-700度             5.44        
          701-1000度            6.16         
          1001度以上            6.71         
          ...................................	  
          **請注意, 電費的計算是以單一費率收費,  
          例如:用電750度, 其電費為:

          6.16 * 750  
          = 4620
          => 4,620元 (若有小數, 以四捨五入計)  
```

#### 程式名稱: m02.py (自行完成)
``` python
(1) 類別名稱: Fee
(2) 靜態方法名稱: separate(degree)
    [2-1] 傳入參數: 用電度數(int) 
    [2-2] 回傳值: 電費(int)
    [2-3] 說明: 
          註:
          ...................................
                               每度（元）   
          ...................................
          330度(含)以下         3.02               
          331-700度             5.44        
          701-1000度            6.16         
          1001度以上            6.71         
          ...................................	  
          **請注意, 電費的計算是各個用電階段以不同計價方式階段計算, 
          例如:用電750度, 其電費為:

          3.02*330 + 5.44*370 + 6.16*50
          => 3,317元 (小數四捨五入)
```
