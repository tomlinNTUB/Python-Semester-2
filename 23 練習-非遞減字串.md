## 1. 非遞減字串-檢查

#### 程式名稱: hw23_1.py
``` python
(1) 建立一個[非遞減字串]類別
    (1-1) 類別名稱: NonDec
    (1-2) 建構元: 傳入[編號](no), [待檢查字串](string) 等2個參數.
                  將參數存入實體中, 成為實體的成員.
                  
    (1-3) [檢查]方法, check(self).
          [1] 假設self.num的內容都是由'0'~'9'組成, 而且它的內容至少有2個字.
          [2] self.num中除了第1個字的每個字, 都和前面的字相同或更大, 就回傳True, 否則回傳False.
          
          例如: '111222333' ==> True
                '111333222' ==> False
          
(2) 產生實體
(3) 顯示資料
```

#### 測試
``` python
1, '1112223334444555566667777'
2, '3337777899922222222222222222'
3, '3334555566777788999999'
4, '11133336666777788881111'
5, '1111222111111222222777777888888899999'
6, '22223333444499999999'
```

#### 輸出
``` python
編號:1, 是否非遞減:True
編號:2, 是否非遞減:False
編號:3, 是否非遞減:True
編號:4, 是否非遞減:False
編號:5, 是否非遞減:False
編號:6, 是否非遞減:True
```



## 2. 非遞減字串-最長非遞減長度

#### 程式名稱: hw23_1.py
``` python
(1) 建立一個[非遞減字串]類別
    (1-1) 類別名稱: NonDec
    (1-2) 建構元: 傳入[編號](no), [待檢查字串](string) 等2個參數.
                  將參數存入實體中, 成為實體的成員.
                  
    (1-3) [最長非遞減內容長度]方法, maxlen(self).
          [1] [每個字串]都包括[非遞減字串], 有時那個字串本身就是一個非遞減字串.
          [2] 請回傳字串中[最長的非遞減字串]的長度.
          
          例如: '112233' ==> 6
                (註:'112233')
                
                '44113322' ==> 4
                (註:'1133')
          
(2) 產生實體
(3) 顯示資料
```

#### 測試
``` python
1, '1112223334444555566667777'
2, '3337777899922222222222222222'
3, '3334555566777788999999'
4, '11133336666777788881111'
5, '1111222111111222222777777888888899999'
6, '22223333444499999999'
```

#### 輸出
``` python
編號:1, 最長非遞減長度:25
編號:2, 最長非遞減長度:17
編號:3, 最長非遞減長度:22
編號:4, 最長非遞減長度:19
編號:5, 最長非遞減長度:30
編號:6, 最長非遞減長度:20
```
