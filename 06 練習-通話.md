### 計算間隔秒數的範例

``` python
#---------------------------------------
# 以下的程式片段可以計算2個時間的間隔秒數
#---------------------------------------
from datetime import datetime

start = '2020-3-9 14:30:15'
end = '2020-3-9 14:32:19'

s = datetime.strptime(start, "%Y-%m-%d %H:%M:%S")
e = datetime.strptime(end, "%Y-%m-%d %H:%M:%S")  

seconds = (s - e).total_seconds()
```
    
#### 測試
``` python
'0911-111111', '2019-12-20 9:0:0', '2019-12-20 9:0:36'
'0911-222222', '2019-12-20 9:0:0', '2019-12-20 9:2:23'
'0911-333333', '2019-12-20 9:0:1', '2019-12-20 9:0:21'
'0911-444444', '2019-12-20 9:0:1', '2019-12-20 9:3:42'
```

#### 輸出
``` python
>> 手機號碼:0911-111111, 通話時間:36秒
>> 手機號碼:0911-222222, 通話時間:143秒
>> 手機號碼:0911-333333, 通話時間:20秒 
>> 手機號碼:0911-444444, 通話時間:221秒
```
