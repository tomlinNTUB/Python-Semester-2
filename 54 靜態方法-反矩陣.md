## 2. 矩陣運算-反矩陣

#### 程式名稱: m02.py (自行完成)
``` python
(1) 類別名稱: Matrice
(2) 靜態方法名稱: inverse(m)
    [2-1] 傳入參數: 一個(2*2)矩陣.
    [2-2] 回傳值: 反矩陣
    [2-3] 說明:  
          如果有一個矩陣M1, 其反矩陣為M2, 則:
          M1 * M2 = I  
          (註:I代表主對角線為1, 其他為0的矩陣, 如:
              [1, 0, 0]
              [0, 1, 0]
              [0, 0, 1] 
              就是一個I.)
    
          反矩陣算法:
          (1) 如果傳入矩陣為:
              [a, b],
              [c, d]
          
          (2) 若其det值為det,
          
          (3) 其反矩陣:
              [ d/det, -b/det],
              [-c/det,  a/det]
          
          
          例如:
          (1) 如果傳入矩陣為:
              [5, 3]
              [2, 1]
          
          (2) 其det值 = 5*1 - 3*2 = -1
          
          (3) 其反矩陣=
              [1/-1, -3/-1]
              [-2/-1, 5/-1]
          
              等於
              [-1, 3]
              [2, -5]          
```
