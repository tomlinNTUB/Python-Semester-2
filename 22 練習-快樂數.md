## 1. 快樂數-分拆數字

#### 程式名稱: hw22_1.py
``` python
(1) 函式名稱: divide(int)
(2) 傳入參數: 一個正整數.
(3) 回傳值: 依傳入參數的各個位數, 依序存在一個清單中再回傳.
            例如: 傳入參數是362, 請回傳 [3, 6, 2].

```


## 2. 快樂數-檢查是否快樂數

#### 程式名稱: hw22_2.py
``` python
(1) 函式名稱: happy(int)
(2) 傳入參數: 一個正整數.
(3) 回傳值: 如果傳入參數是快樂數, 回傳true;
           如果不是, 回傳false.
            
(4) 檢查是否快樂數的方法.     
     [傳入參數] 的所有數位之平方和，得到的新數再次求所有數位的平方和，如此重複進行，如果其中的平方和之值出現為1, 就是[快樂數].           
           例如: 28和32是快樂數, 原因如下:
           
           (1) 28 -> 2*2 + 8*8 = 68 
                  -> 6*6 + 8*8 = 100 
                  -> 1*1 + 0*0 + 0*0 = 1 (快樂數)
              
           (2) 32 -> 3*3 + 2*2 = 13
                  -> 1*1 + 3*3 = 10
                  -> 1*1 + 0*0 = 1 (快樂數)
                  
           如果在求所有數位的平方和的過程中, 出現過去已出現的平方和, 表示再算下去也會一直循環, 它不是快樂數.
           例如: 15和21不是快樂數, 原因如下:
           
           (1) 15 -> 1*1 + 5*5 = 26
                  -> 2*2 + 6*6 = 40 
                  -> 4*4 + 0*0 = 16
                  -> 1*1 + 6*6 = 37
                  -> 3*3 + 7*7 = 58
                  -> 5*5 + 8*8 = 89
                  -> 8*8 + 9*9 = 145
                  -> 1*1 + 4*4 + 5*5 = 42
                  -> 4*4 + 2*2 = 20
                  -> 2*2 = 4
                  -> 4*4  = 16 (與第3次平方和相同, 不是快樂數)
            
           (2) 21 -> 2*2 + 1*1 = 5
                  -> 5*5 = 25
                  -> 2*2 + 5*5 = 29
                  -> 2*2 + 9*9 = 85
                  -> 8*8 + 5*5 = 89
                  -> 8*8 + 9*9 = 145
                  -> 1*1 + 4*4 + 5*5 = 42
                  -> 4*4 + 2*2 = 20
                  -> 2*2 = 4
                  -> 4*4  = 16 
                  -> 1*1 + 6*6 = 37
                  -> 3*3 + 7*7 = 58
                  -> 5*5 + 8*8 = 89  (與第4次平方和相同, 不是快樂數)
```

#### 測試
``` python
1, 32    
2, 97 
3, 139 
4, 362 
5, 228 
6, 399
```

#### 輸出
``` python
編號:1, 是否快樂數:True
編號:2, 是否快樂數:True
編號:3, 是否快樂數:True
編號:4, 是否快樂數:True
編號:5, 是否快樂數:False
編號:6, 是否快樂數:False
```
