## 1. 學生類別

#### 程式名稱: hw02_1.py
``` python
(1) 建立一個[學生]類別
    (1-1) 類別名稱: Student
    (1-2) 建構元: 傳入[學號](no)及[姓名](name)2個參數, 寫入實體中, 成為實體的成員(屬性).
    (1-3) 方法: 部別, division()
          回傳學號所屬的部別, 例如:學號10856001的第4碼是[5]則回傳[五專]; [3]回傳[二技]; [4]回傳[四技].

(2) 產生實體

(3) 顯示實體的成員值.
```

#### 測試
``` python
'10856001', '王小明'
'10846023', '陳小華'
'10836045', '張小文'
```

#### 輸出
``` python
>> 10856001
>> 王小明
>> 五專
>> 10846023
>> 陳小華
>> 四技
>> 10836045
>> 張小文
>> 二技
```
