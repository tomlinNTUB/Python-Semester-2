#### 程式名稱: hw33_1.py
``` python
問題: 在'吃葡萄不吐葡萄皮'的所有**內容不同的**排列中, 有多少個[葡萄]2字是相連在一起, 沒有被分開或顛倒位置?
```

#### 參考答案
``` python
  .
  .
  .
('葡', '萄', '葡', '萄', '不', '吃', '吐', '皮')
('葡', '萄', '皮', '葡', '萄', '不', '吐', '吃')

('皮', '不', '葡', '萄', '吃', '吐', '葡', '萄')
```

## 2. 絕對[不吃皮]

#### 程式名稱: hw33_2.py
``` python

     (註: 這題與前一題無關, 不管是否有2個葡萄的連字.)
```

#### 參考答案
``` python
  .
  .
  .
('萄', '不', '吃', '皮', '萄', '葡', '吐', '葡')
('葡', '葡', '不', '吃', '皮', '萄', '萄', '吐')
('萄', '葡', '葡', '萄', '吐', '不', '吃', '皮')
('萄', '葡', '吐', '葡', '萄', '不', '吃', '皮')
('葡', '不', '吃', '皮', '萄', '吐', '萄', '葡')
('葡', '吐', '萄', '葡', '不', '吃', '皮', '萄')
('萄', '萄', '不', '吃', '皮', '吐', '葡', '葡')
```
