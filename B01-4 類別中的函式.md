# B01-4 類別中的函式


## 1. 成績類別內計算總分與平均的函式 

#### 1-1 學習重點
``` python
(1) 類別內的[函式]主要是為了處理類別內的屬性資料而存在, 將[資料]與[處理方法]封裝在一個[類別]中, 增加類別的可重覆使用性.
```

#### 1-2 程式範例

``` python
#---------------------------
# 成績[類別]
#---------------------------
class Score():
    # 建構元
    def __init__(self, chi, eng, mat):
        self.chi = chi
        self.eng = eng
        self.mat = mat

    # 總分
    def total(self):
        return self.chi + self.eng + self.mat

    # 平均
    def average(self):
        return self.total() / 3

#===========================
# 主程式
#===========================
# 建立3個成績[物件]
s1 = Score(80, 90, 70)
s2 = Score(75, 60, 75)
s3 = Score(70, 95, 68)

# 印出物件內容
print('總分:{}'.format(s1.total()))
print('平均:{:.2f}'.format(s1.average()))
print()

print('總分:{}'.format(s2.total()))
print('平均:{:.2f}'.format(s2.average()))
print()

print('總分:{}'.format(s3.total()))
print('平均:{:.2f}'.format(s3.average()))
```
