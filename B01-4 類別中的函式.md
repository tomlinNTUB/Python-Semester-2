# B01-4 類別中的函式


## 1. 成績類別內計算總分與平均的函式 

#### 1-1 學習重點
``` python
(1) 類別內的[函式]主要是為了處理類別內的屬性資料而存在, 
    將[資料]與[處理方法]封裝在一個[類別]中, 增加類別的可重覆使用性.
```

#### 1-2 程式範例

``` python
#---------------------------
# 成績[類別]
#---------------------------
class Score():
    # 建構元
    def __init__(self, chi, eng, mat):
        self.chi = chi
        self.eng = eng
        self.mat = mat

    # 總分
    def total(self):
        return self.chi + self.eng + self.mat

    # 平均
    def average(self):
        return self.total() / 3

#===========================
# 主程式
#===========================
# 建立3個成績[物件]
s1 = Score(80, 90, 70)
s2 = Score(75, 60, 75)
s3 = Score(70, 95, 68)

# 印出物件內容
print('總分:{}'.format(s1.total()))
print('平均:{:.2f}'.format(s1.average()))
print()

print('總分:{}'.format(s2.total()))
print('平均:{:.2f}'.format(s2.average()))
print()

print('總分:{}'.format(s3.total()))
print('平均:{:.2f}'.format(s3.average()))
```

#### 1-3 執行結果
``` python
>> 總分:240
>> 平均:80.00
>> 
>> 總分:210
>> 平均:70.00
>> 
>> 總分:233
>> 平均:77.67
```


## 2. 增加一個判斷等級的函式 


#### 2-1 程式範例

``` python
#---------------------------
# 成績[類別]
#---------------------------
class Score():
    # 建構元
    def __init__(self, chi, eng, mat):
        self.chi = chi
        self.eng = eng
        self.mat = mat

    # 總分
    def total(self):
        return self.chi + self.eng + self.mat

    # 平均
    def average(self):
        return self.total() / 3

    # 等級
    def rank(self):
        a = self.average()

        if a>=90:
            return '優等'
        elif a>=80:
            return '甲等'
        elif a>=70:
            return '乙等'
        elif a>=60:
            return '丙等'
        else:
            return '丁等'         

#===========================
# 主程式
#===========================
# 建立3個成績[物件]
s1 = Score(80, 90, 70)
s2 = Score(55, 60, 75)
s3 = Score(70, 95, 68)

# 印出物件內容
print('總分:{}'.format(s1.total()))
print('平均:{:.2f}'.format(s1.average()))
print('等級:{}'.format(s1.rank()))
print()

print('總分:{}'.format(s2.total()))
print('平均:{:.2f}'.format(s2.average()))
print('等級:{}'.format(s2.rank()))
print()

print('總分:{}'.format(s3.total()))
print('平均:{:.2f}'.format(s3.average()))
print('等級:{}'.format(s3.rank()))
```

#### 2-2 執行結果
``` python
>> 總分:240
>> 平均:80.00
>> 等級:甲等
>> 
>> 總分:190
>> 平均:63.33
>> 等級:丙等
>> 
>> 總分:233
>> 平均:77.67
>> 等級:乙等
```
