# B01-5 類別與主程式分別儲存


## 1. 將成績類別存在m01.py中 

#### 1-1 學習重點
``` python
(1) 儲存架構
      |__ m01.py
      |__ p01.py

(2) 使用import指令匯入模組.
```

#### 1-2 程式範例(m01.py)

``` python
#---------------------------
# 成績[類別]
#---------------------------
class Score():
    # 建構元
    def __init__(self, chi, eng, mat):
        self.chi = chi
        self.eng = eng
        self.mat = mat

    # 總分
    def total(self):
        return self.chi + self.eng + self.mat

    # 平均
    def average(self):
        return self.total() / 3

    # 等級
    def rank(self):
        a = self.average()

        if a>=90:
            return '優等'
        elif a>=80:
            return '甲等'
        elif a>=70:
            return '乙等'
        elif a>=60:
            return '丙等'
        else:
            return '丁等'         
```


#### 1-3 程式範例(p01.py)

``` python
# 匯入模組中的類別
from m01 import Score

# 建立3個成績[物件]
s1 = Score(80, 90, 70)
s2 = Score(55, 60, 75)
s3 = Score(70, 95, 68)

# 印出物件內容
print('總分:{}'.format(s1.total()))
print('平均:{:.2f}'.format(s1.average()))
print('等級:{}'.format(s1.rank()))
print()

print('總分:{}'.format(s2.total()))
print('平均:{:.2f}'.format(s2.average()))
print('等級:{}'.format(s2.rank()))
print()

print('總分:{}'.format(s3.total()))
print('平均:{:.2f}'.format(s3.average()))
print('等級:{}'.format(s3.rank()))
```
