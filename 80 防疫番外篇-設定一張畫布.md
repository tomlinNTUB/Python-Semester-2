## 1-3. 程式 (p01.py)

``` python
#---------------------
# 匯入模組
#---------------------
import sys
import pygame
from pygame.locals import QUIT

#=====================
# 主程式
#=====================
# 啟動pygame
pygame.init()

# 設定畫面尺寸及標題
screen_surface = pygame.display.set_mode((800, 600))
pygame.display.set_caption('資一甲好學生大亂鬥')

# 產生時間物件, 用來設定切換畫面頻率
clock = pygame.time.Clock()

# 遊戲主流程, 無窮迴圈
while True:
    # 偵測事件
    for event in pygame.event.get():
        # 結束
        if event.type == QUIT:
            pygame.quit()
            sys.exit()               
    
    # 背景顏色, 也清除畫面
    screen_surface.fill((200, 200, 200))

    # 更新畫面
    pygame.display.update()

    # 遊戲速率(每秒畫面更新次數)
    clock.tick(100)
```
