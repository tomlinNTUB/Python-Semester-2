``` python
                              Food 
                          (食物, 父親)
                                ^
                                | (繼承)
        |-----------------------|----------------------|
        |                       |                      |
      Pizza                   Wulong                 Cookie
    (披薩, 兒子)           (烏龍麵, 兒子)           (餅乾, 兒子)     
   
  
註:
1. 「披薩」的「訂價」計算方式:
    (1) 如果是「手工製作」:
        (a) 如果尺寸是'S', 訂價350元;
        (b) 如果尺寸是'M', 訂價450元;
        (c) 如果尺寸是'L', 訂價600元;
       
    (2) 如果是「不是手工製作」:
        (a) 如果尺寸是'S', 訂價250元;
        (b) 如果尺寸是'M', 訂價350元;
        (c) 如果尺寸是'L', 訂價500元;
       
   (3) 如果「項目名稱」中包括:
        (a) '海鮮', 加價100;
        (b) '豪華', 加價150;
        (c) '無敵', 加價150;
       
        *請注意, 有些項目可能包括以上2個或3個內容, 
         如: '豪華海鮮', 此時, 加價 100+150 = 250元.

   (4) 如果「加多起司」, 加價90元. 
   
   (5) 如果「加多配料」, 加價100元.
   
   
2. 「烏龍麵」的「訂價」計算方式:
    (1) 如果是「手工製作」:
        (a) 如果尺寸是'M', 訂價70元;
        (b) 如果尺寸是'L', 訂價80元;
       
    (2) 如果是「不是手工製作」:
        (a) 如果尺寸是'M', 訂價50元;
        (b) 如果尺寸是'L', 訂價60元;   
       
    (3) 如果「炸物追加」, 前3項每樣10元, 後2項每樣15元.       


3. 「餅乾」的「訂價」計算方式:
    (1) 如果是「手工製作」, 每個單價45元.
    
    (2) 如果是「不是手工製作」, 每個單價15元:
    
    (3) 訂價 = 每個單價 * 數量.
    
    (4) 如果是「手工製作」, 有以下折扣:
        (a) 如果數量>30, 打85折;
        (b) 如果數量>50, 打75折.
```

``` python
import abc

#-------------------------
# 食物, 父類別(已完成)
#-------------------------
class Food(abc.ABC):
    # 傳入項目名稱, 尺寸, 是否手工製作
    def __init__(self, title, size, isHandMade):
        self.title = title+'('+size+')'
        if isHandMade:
            self.title+='(手工製作)'
        else:
            self.title+='(機器製作)'
        self.size = size
        self.isHandMade = isHandMade

    # 訂價, 抽象方法  
    @abc.abstractmethod
    def price(self):
        return NotImplemented
```
