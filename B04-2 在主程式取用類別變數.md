# B04-2 在主程式取用類別變數

## 1. 在主程式, 可用[類別名稱]取用該類別的[類別變數]

#### 1-1 學習重點
``` python
(1) 儲存架構
      |__ m01.py
      |__ p01.py
      
(2) 在類別之外, 也可以取用類別變數.
    可用:類別名稱.類別變數名稱, 如:
    Score.w_chi
```

#### 1-2 程式範例(m01.py)

``` python
#---------------------------
# 成績[類別]
#---------------------------
class Score():
    # 各科權重(類別變數)
    w_chi = 1.5
    w_eng = 2
    w_mat = 1

    # 建構元
    def __init__(self, chi, eng, mat):
        self.chi = chi
        self.eng = eng
        self.mat = mat

    # 總分
    def total(self):
        return self.chi*Score.w_chi + self.eng*Score.w_eng + self.mat*Score.w_mat

    # 平均
    def average(self):
        return self.total() / (Score.w_chi+Score.w_eng+Score.w_mat)
```


#### 1-3 程式範例(p01.py)

``` python
# 匯入模組中的類別
from m01 import Score

# 建立儲存成績物件的list
scores = []

scores.append(Score(60, 80, 90))
scores.append(Score(65, 65, 70))
scores.append(Score(85, 70, 85))

# 印出類別變數(以類別名稱取用)
print('國文加權:{}'.format(Score.w_chi))
print('英文加權:{}'.format(Score.w_eng))
print('數學加權:{}'.format(Score.w_mat))
print()

# 印出物件內容
for s in scores:
    print('總分:{}'.format(s.total()))
    print('平均:{:.2f}'.format(s.average()))
    print()
```


#### 1-4 執行結果
``` python

>> 英文加權:2
>> 數學加權:1
>> 
>> 總分:340.0
>> 平均:75.56
>> 
>> 總分:297.5
>> 平均:66.11
>> 
>> 總分:352.5
>> 平均:78.33
>> 
```


## 2. 在主程式, 用[物件名稱]也可能可以取用該類別的[類別變數]

#### 2-1 學習重點
``` python
(1) 儲存架構
      |__ m01.py
      |__ p01.py
      
(2) 用:物件名稱.類別變數名稱, 也有可能會取用到[類別變數](前題是此物件沒有同名的物件變數):
    scores[0].w_chi
    
    應注意, 它取用的順序是:
    (3-1) 先找此物件有沒有名稱是w_chi的[物件變數](該物件專有的屬性)? 如果有, 就使用; 
    (3-2) 如果沒有, 再找此物件所屬類別的[類別變數].
```

#### 2-2 程式範例(m01.py)

``` python
#---------------------------
# 成績[類別]
#---------------------------
class Score():
    # 各科權重(類別變數)
    w_chi = 1.5
    w_eng = 2
    w_mat = 1

    # 建構元
    def __init__(self, chi, eng, mat):
        self.chi = chi
        self.eng = eng
        self.mat = mat

    # 總分
    def total(self):
        return self.chi*Score.w_chi + self.eng*Score.w_eng + self.mat*Score.w_mat

    # 平均
    def average(self):
        return self.total() / (Score.w_chi+Score.w_eng+Score.w_mat)
```


#### 2-3 程式範例(p01.py)

``` python
# 匯入模組中的類別
from m01 import Score

# 建立儲存成績物件的list
scores = []

scores.append(Score(60, 80, 90))
scores.append(Score(65, 65, 70))
scores.append(Score(85, 70, 85))

# 印出類別變數(以物件名稱取用)
print('國文加權:{}'.format(scores[0].w_chi))
print('英文加權:{}'.format(scores[0].w_eng))
print('數學加權:{}'.format(scores[0].w_mat))
print()

# 印出物件內容
for s in scores:
    print('總分:{}'.format(s.total()))
    print('平均:{:.2f}'.format(s.average()))
    print()
```
