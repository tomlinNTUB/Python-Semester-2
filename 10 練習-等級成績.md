## 1. 等級成績-計算總分

#### 程式名稱: hw10_1.py
``` python
(1) 建立一個[等級成績]類別
    (1-1) 類別名稱: RankScore
    (1-2) 建構元: 傳入[編號](no), [等級](*rank) 等2個參數, 其中[等級]將存入4個參數值.
                  將參數存入實體中, 成為實體的成員.
    (1-3) [總分]方法, total().
          (1) 等級A+, 可得5分;
              等級A,  可得4分;
              等級A-, 可得3分;
              等級B+, 可得2分;
              等級B,  可得1分;
              其他,  得0分.
          (2) 總分 = 4次等級相對得分相加.

(2) 產生實體
(3) 顯示資料
```

#### 測試
``` python
1, 'A', 'A-', 'A', 'B+' 
2, 'B-', 'C', 'B', 'B-' 
3, 'B+', 'B', 'B', 'B+'
4, 'C', 'C', 'C', 'C'
5, 'A+', 'A+', 'A', 'A+' 
```

#### 輸出
``` python
編號:1, 總分:13
編號:2, 總分:1
編號:3, 總分:6
編號:4, 總分:0
編號:5, 總分:19
```


## 2. 等級成績-是否通過

#### 程式名稱: hw10_2.py
``` python
(1) 建立一個[等級成績]類別
    (1-1) 類別名稱: RankScore
    (1-2) 建構元: 傳入[編號](no), [等級](*rank) 等2個參數, 其中[等級]將存入4個參數值.
                  將參數存入實體中, 成為實體的成員.
    (1-3) [總分]方法, total().
          (1) 等級A+, 可得5分;
              等級A,  可得4分;
              等級A-, 可得3分;
              等級B+, 可得2分;
              等級B,  可得1分;
              其他,  得0分.
          (2) 總分 = 4次等級相對得分相加.
          
    (1-4) [是否通過]方法, isPass().
          若: [總分<12] 或 [考了至少1個'C'], 回傳False; 否則回傳True.
          
(2) 產生實體
(3) 顯示資料
```

#### 測試
``` python
1, 'A', 'A-', 'A', 'B+' 
2, 'B-', 'C', 'B', 'B-' 
3, 'B+', 'B', 'B', 'B+'
4, 'C', 'C', 'C', 'C'
5, 'A+', 'A+', 'A', 'A+' 
```

#### 輸出
``` python
編號:1, 是否通過:True
編號:2, 是否通過:False
編號:3, 是否通過:False
編號:4, 是否通過:False
編號:5, 是否通過:True
```
