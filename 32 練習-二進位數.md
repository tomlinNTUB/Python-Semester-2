## 1. 二進位數-長度20的二進位數

#### 程式名稱: hw32_1.py
``` python
(1) 建立一個[二進位數]類別
    (1-1) 類別名稱: Binary
    (1-2) 建構元: 傳入[編號](no), [十進位數](num) 等2個參數.
                  將參數存入實體中, 成為實體的成員.
                  
    (1-3) 長度20的二進位數: bin(self), 將self.num轉成二進位數, 若不足20位, 補足前導0.
          假設0 <= self.num <= 1000000, 
          回傳型態是字串.         
          
     
(2) 產生實體
(3) 顯示資料
```

#### 測試
``` python
1, 50
2, 100
3, 1024
4, 30000
5, 1000000
```

#### 輸出
``` python
編號:1, 二進位碼:00000000000000110010
編號:2, 二進位碼:00000000000001100100
編號:3, 二進位碼:00000000010000000000
編號:4, 二進位碼:00000111010100110000
編號:5, 二進位碼:11110100001001000000
```



## 2. 二進位數-漢明距離

#### 程式名稱: hw32_2.py
``` python
(1) 建立一個[二進位數]類別
    (1-1) 類別名稱: Binary
    (1-2) 建構元: 傳入[編號](no), [十進位數](num) 等2個參數.
                  將參數存入實體中, 成為實體的成員.
                  
    (1-3) 長度20的二進位數: bin(self), 將self.num轉成二進位數, 若不足20位, 補足前導0.
          假設0 <= self.num <= 1000000, 
          回傳型態是字串.         

    (1-4) 漢明距離: hammingDistance(self), 計算self.num與二進位數'11110000111100001111'的漢明距離.
          註: 2個數的漢明距離 = 2個數的二進位數在相同位置的值不相同的個數.
              例: '100110' 和
                  '101011'              
                  從左至右, 有第3, 4, 6的位置值不同, 共有3個不同, 漢明距離=3.              
               
(2) 產生實體
(3) 顯示資料
```

#### 測試
``` python
1, 50
2, 100
3, 1024
4, 30000
5, 1000000
```

#### 輸出
``` python
編號:1, 漢明距離:13
編號:2, 漢明距離:13
編號:3, 漢明距離:11
編號:4, 漢明距離:15
編號:5, 漢明距離:9
```
