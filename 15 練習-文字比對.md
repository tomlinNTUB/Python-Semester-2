## 1. 文字比對-多出的文字

#### 程式名稱: hw15_1.py
``` python
(1) 建立一個[文字比對]類別
    (1-1) 類別名稱: Compare
    (1-2) 建構元: 傳入[編號](no), [文字1](str1), [文字2](str2), 等3個參數.
                  將參數存入實體中, 成為實體的成員.
    (1-3) [找出多出來的文字]方法, extra().
          註:
          [文字2]的內容是[文字1]的內容重新排列的結果, 再加1個文字.
          請找出[文字2]多出的那個文字.  
          例如: [文字1]的內容是 'gkumoi'
                [文字2]的內容是 'kgihuom'
                多出的那個文字是'h'
          
(2) 產生實體
(3) 顯示資料
```

#### 測試
``` python
1, 'gkumoi', 'kgihuom'
2, 'ymperb', 'mqrbeyp'
3, 'wpiqztg', 'iqwpkgtz'
4, 'gkorqxzip', 'qxoriapzkg'
5, 'hmopedx', 'mpkoxdeh'
```

#### 輸出
``` python
編號:1, 多出的字:h
編號:2, 多出的字:q
編號:3, 多出的字:k
編號:4, 多出的字:a
編號:5, 多出的字:k
```


## 2. 文字比對-多出的文字

#### 程式名稱: hw15_2.py
``` python
(1) 建立一個[文字比對]類別
    (1-1) 類別名稱: Compare
    (1-2) 建構元: 傳入[編號](no), [文字1](str1), [文字2](str2), 等3個參數.
                  將參數存入實體中, 成為實體的成員.
    (1-3) [找出多出來的文字]方法, extra().
          註:
          [文字1]和[文字2]有以下的可能:
          
          [1] 內容的組成文字都一樣, 只是順序不同.
          [2] [文字1]比[文字2]多1個字, 其他的內容組成都一樣, 只是順序不同.
          [3] [文字2]比[文字1]多1個字, 其他的內容組成都一樣, 只是順序不同.
          
          請找出它們的關係:
          如果是上述的情況[1], 回傳'二者相同';  
          如果是上述的情況[2]或[3], 回傳多出的那個文字.
          
(2) 產生實體
(3) 顯示資料
```

#### 測試
``` python
1, 'gkumoi', 'kgihuom'
2, 'mqrbeyp', 'ymperb'
3, 'wpiqztg', 'iqwpkgtz'
4, 'qxoriapzkg', 'gkorqxzip'
5, 'hmopedx', 'mpoxdeh'
```

#### 輸出
``` python
編號:1, 多出的字:h
編號:2, 多出的字:q
編號:3, 多出的字:k
編號:4, 多出的字:a
編號:5, 多出的字:二者相同
```
