## 1. 電費-單一費率

#### 程式名稱: hw13_1.py
``` python
(1) 建立一個[電費]類別
    (1-1) 類別名稱: ElecFee
    (1-2) 建構元: 傳入[編號](no), [用電度數](degree) 等2個參數.
                  將參數存入實體中, 成為實體的成員.
    (1-3) [電費]方法, fee().
          註:
          ...................................
                               每度（元）   
          ...................................
          120度(含)以下         2.10          
          121-330度             3.02        
          331-500度             4.39         
          501-700度             5.44        
          701-1000度            6.16         
          1001度以上            6.71         
          ...................................	  
          **請注意, 電費的計算是以單一費率收費,  
          例如:用電750度, 其電費為:

          6.16 * 750  
          = 4620
          => 4,620元 (若有小數, 以四捨五入計)  

(2) 產生實體
(3) 顯示資料
```

#### 測試
``` python
1, 100
2, 200
3, 400
4, 600
5, 800
6, 1200
```

#### 輸出
``` python
編號:1, 電費:210元
編號:2, 電費:604元
編號:3, 電費:1,756元
編號:4, 電費:3,264元
編號:5, 電費:4,928元
編號:6, 電費:8,052元
```


## 2. 電費-累進費率

#### 程式名稱: hw13_2.py
``` python
(1) 建立一個[電費]類別
    (1-1) 類別名稱: ElecFee
    (1-2) 建構元: 傳入[編號](no), [用電度數](degree) 等2個參數.
                  將參數存入實體中, 成為實體的成員.
    (1-3) [電費]方法, fee().
          註:
          ...................................
                                  每度（元）   
          ...................................
          120度(含)以下部分         2.10          
          121-330度部分            3.02        
          331-500度部分             4.39         
          501-700度部分             5.44        
          701-1000度部分            6.16         
          1001度以上 部分           6.71         
          ...................................	  
          **請注意, 電費的計算是各個用電階段以不同計價方式階段計算, 
          例如:用電750度, 其電費為:

          2.10*120 + 3.02*210 + 4.39*170 + 5.44*200 + 6.16*50
          = 3028.5
          => 3,029元 (小數四捨五入)

(2) 產生實體
(3) 顯示資料
```

#### 測試
``` python
1, 100
2, 200
3, 400
4, 600
5, 800
6, 1200
```

#### 輸出
``` python
編號:1, 電費:210元
編號:2, 電費:494元
編號:3, 電費:1,194元
編號:4, 電費:2,176元
編號:5, 電費:3,336元
編號:6, 電費:5,910元
```
