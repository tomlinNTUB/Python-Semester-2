## 1. 學生類別

#### 程式名稱: hw01_1.py
``` python
(1) 建立一個[學生]類別
    (1-1) 類別名稱: Student
    (1-2) 建構元: 傳入[學號](no)及[姓名](name)2個參數, 寫入實體中, 成為實體的成員(屬性).

(2) 產生2個學生實體, s1及s2

(3) 顯示實體的成員值.
```

#### 測試
``` python
'120001', '王小明'
'120002', '陳小華'
```

#### 輸出
``` python
>> 120001
>> 王小明
>> 120002
>> 陳小華'
```
