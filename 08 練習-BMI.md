## 1. BMI-BMI值

#### 程式名稱: hw08_1.py
``` python
(1) 建立一個[稅金]類別
    (1-1) 類別名稱: BMI
    (1-2) 建構元: 傳入[姓名](name), [身高](height), [體重](weight)等3個參數, 寫入實體中, 成為實體的成員.
                 其中, [身高以公分為單位], [體重以公斤為單位].
    (1-3) [計算BMI值]方法, value(), 
          其值 = 體重 / (身高的平方), 其中, [身高以公尺為單位], [體重以公斤為單位].          

(2) 產生實體
(3) 顯示資料, bmi值顯示至小數第2位, 其後四捨五入.
```

#### 測試
``` python
'王小明', 175, 78  
'陳小華', 163, 50 
'張小文', 178, 51
'李小仁', 163, 77    
'周小全', 156, 43   
```

#### 輸出
``` python
姓名:王小明, bmi值:25.47
姓名:陳小華, bmi值:18.82
姓名:張小文, bmi值:16.10
姓名:李小仁, bmi值:28.98
姓名:周小全, bmi值:17.67
```

## 2. BMI-評估

#### 程式名稱: hw08_2.py
``` python
(1) 建立一個[稅金]類別
    (1-1) 類別名稱: BMI
    (1-2) 建構元: 傳入[姓名](name), [身高](height), [體重](weight)等3個參數, 寫入實體中, 成為實體的成員.
                 其中, [身高以公分為單位], [體重以公斤為單位].
    (1-3) [計算BMI值]方法, value(), 
          其值 = 體重 / (身高的平方), 其中, [身高以公尺為單位], [體重以公斤為單位].
    (1-4) [BMI評語]方法, comment(), 
           如果 BMI < 18.5, 評語是"過輕";
           如果 18.5 <= BMI < 24 , "正常";
           如果 24 <= BMI < 27, "過重";
           如果 BMI >= 27, "肥胖".          

(2) 產生實體
(3) 顯示資料, bmi值顯示至小數第2位, 其後四捨五入.
```

#### 測試
``` python
'王小明', 175, 78  
'陳小華', 163, 50 
'張小文', 178, 51
'李小仁', 163, 77    
'周小全', 156, 43   
```

#### 輸出
``` python
姓名:王小明, bmi值:25.47, 評語:過重
姓名:陳小華, bmi值:18.82, 評語:正常
姓名:張小文, bmi值:16.10, 評語:過輕
姓名:李小仁, bmi值:28.98, 評語:肥胖
姓名:周小全, bmi值:17.67, 評語:過輕
```
