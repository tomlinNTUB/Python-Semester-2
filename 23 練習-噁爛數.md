## 1. 噁爛數-質因數

#### 程式名稱: hw23_1.py
``` python
(1) 建立一個[噁爛數]類別
    (1-1) 類別名稱: Ugly
    (1-2) 建構元: 傳入[編號](no), [待檢查數字](num) 等2個參數.
                  將參數存入實體中, 成為實體的成員.
    (1-3) [質因數]方法, prime(self).
          找出self.num的質因數, 放入set中, 再回傳.
          質數: 除了1和自己, 沒有其他整數可以整除它.
          
          例如: 96 = 2 * 2 * 2 * 2 * 2 * 3
                96的質因數包括: {2, 3}
          
(2) 產生實體
(3) 顯示資料
```

#### 測試
``` python
1, 1024    
2, 15 
3, 105
4, 450
5, 245
6, 36
```

#### 輸出
``` python
編號:1, 質因數:{2}
編號:2, 質因數:{3, 5}
編號:3, 質因數:{3, 5, 7}
編號:4, 質因數:{2, 3, 5}
編號:5, 質因數:{5, 7}
編號:6, 質因數:{2, 3}
```


## 2. 噁爛數-檢查

#### 程式名稱: hw23_2.py
``` python
(1) 建立一個[噁爛數]類別
    (1-1) 類別名稱: Ugly
    (1-2) 建構元: 傳入[編號](no), [待檢查數字](num) 等2個參數.
                  將參數存入實體中, 成為實體的成員.
    (1-3) [質因數]方法, prime(self).
          找出self.num的質因數, 放入set中, 再回傳.
          質數: 除了1和自己, 沒有其他整數可以整除它.
          
          例如: 96 = 2 * 2 * 2 * 2 * 2 * 3
          因此96的不重覆質因數為: {2, 3}
          
    (1-4) [檢查]方法, check(self).
          如果self.num的質因數不包括2, 3, 或5以外的數, 就是噁爛數.
          若是, 請回傳True; 否則回傳False.  
          
          例如: {2, 3} --> True
                {2, 5} --> True
                {2, 3, 5} --> True
                {2, 7} -> False
          
(2) 產生實體
(3) 顯示資料
```

#### 測試
``` python
1, 1024    
2, 15 
3, 105
4, 450
5, 245
6, 36
```

#### 輸出
``` python
編號:1, 是否噁爛數:True
編號:2, 是否噁爛數:True
編號:3, 是否噁爛數:False
編號:4, 是否噁爛數:True
編號:5, 是否噁爛數:False
編號:6, 是否噁爛數:True
```
