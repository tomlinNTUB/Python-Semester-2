# B01-1 撰寫類別

# 1. 未用類別

### 1-1 程式範例
``` python
# 建立2個變數存放學生資料
no = '120001'
name = '王小明'

# 印出學生資料
print('學號:' + no)
print('姓名:' + name)  
```

### 1-2 執行結果

``` python
>> 學號:120001
>> 姓名:王小明
```

# 2. 使用類別 

### 2-1 撰寫重點
| 敘述 | 意義 |
|:---------|:------|
| class | 一個類別, 其中可以定義它所有的資料及函式, 將來它可以產生多個物件  |
| &#95;&#95;init&#95;&#95;(self) | 建構元函式, 在init前後有2個底線, self用來指向產生的物件 |


### 2-2 程式範例 (這個程式的可重覆使用性非常弱)
``` python
#---------------------------
# 學生[類別]
#---------------------------
class Student():
    def __init__(self):
        self.no = '120001'
        self.name = '王小明'

#===========================
# 主程式
#===========================
# 建立一個學生[物件]
s = Student()

# 印出物件內容
print('學號:' + s.no)
print('姓名:' + s.name)
```

### 2-3 執行結果
``` python
>> 學號:120001
>> 姓名:王小明
```
