## 1. 羅馬數字-找出負數位置

#### 程式名稱: hw28_1.py
``` python
(1) 建立一個[羅馬數字]類別
    (1-1) 類別名稱: Roman
    (1-2) 建構元: 傳入[編號](no), [羅馬數字](symbols) 等2個參數.
                  將參數存入實體中, 成為實體的成員.
                  
    (1-3) 找出負值位置: negative(self)
          羅馬數字由以下符號及數值組成:
          符號      數值
          --------------
           I          1
           V          5
           X         10
           L         50
           C        100
           D        500
           M       1000
           
          在一個羅馬數字中, 找出有哪些符號, 它們的右邊有比它大的羅馬符號?
          如: 在"XIV"中, "I"的右邊有"V"比自己大, 因此位置1就是答案.
          請以清單型態回傳結果.
         
(2) 產生實體
(3) 顯示資料
```

#### 測試
``` python
1, 'III'            
2, 'IV' 
3, 'IX' 
4, 'LVIII' 
5, 'MCMXCIV' 
```

#### 輸出
``` python
編號:1, 負數位數:[]
編號:2, 負數位數:[0]
編號:3, 負數位數:[0]
編號:4, 負數位數:[]
編號:5, 負數位數:[1, 3, 5]
```



## 2. 羅馬數字-算出數值

#### 程式名稱: hw28_2.py
``` python
(1) 建立一個[羅馬數字]類別
    (1-1) 類別名稱: Roman
    (1-2) 建構元: 傳入[編號](no), [羅馬數字](symbols) 等2個參數.
                  將參數存入實體中, 成為實體的成員.
                  
    (1-3) 找出負值位置: negative(self)
          羅馬數字由以下符號及數值組成:
          符號      數值
          --------------
           I          1
           V          5
           X         10
           L         50
           C        100
           D        500
           M       1000
           
          在一個羅馬數字中, 找出有哪些符號, 它們的右邊有比它大的羅馬符號?
          如: 在"XIV"中, "I"的右邊有"V"比自己大, 因此位置1就是答案.
          請以清單型態回傳結果.
          
    (1-4) 算出數值: value(self)
          羅馬數字中的每個符號代表一個值, 如上表:
          一個羅馬數字的數值, 就是其中各個羅馬符號代表的值的總和; 但如果該羅馬符號的右邊有比它大的符號, 該符號的值應以負數計算.
          
          如: "XIV" = 10 + (-1) + 5 = 14.          
         
(2) 產生實體
(3) 顯示資料
```

#### 測試
``` python
1, 'III'            
2, 'IV' 
3, 'IX' 
4, 'LVIII' 
5, 'MCMXCIV' 
```

#### 輸出
``` python
編號:1, 數值:3
編號:2, 數值:4
編號:3, 數值:9
編號:4, 數值:58
編號:5, 數值:1994
```
