## 1. 我們與惡的距離-產生清單

#### 程式名稱: hw25_1.py
``` python
(1) 建立一個[我們與惡的距離]類別
    (1-1) 類別名稱: BadNeighbor
    (1-2) 建構元: 傳入[編號](no), [待檢查字串](string) 等2個參數.
                  將參數存入實體中, 成為實體的成員.
                  
    (1-3) [產生惡清單]方法, genlist(self).
          產生一個清單, 它的元素個數與self.string長度一樣.
          清單中的元素設定為999, 當它與self.string中相同順序的元素為'我';
          清單中的元素設定為0, 當它與self.string中相同順序的元素為'惡'.
          
          註: self.string中只有'我'和'惡'2種內容, 長度最多100個字, 至少1個字.
          
          例如: '我惡我我我惡'
          ==>   [999, 0, 999, 999, 999, 0]
          
(2) 產生實體
(3) 顯示資料
```

``` python
5, '惡我我我我我我我惡我我我我我我我我惡我我我我我我我惡'
```

## 2. 我們與惡的距離-計算距離

#### 程式名稱: hw25_2.py
``` python
(1) 建立一個[我們與惡的距離]類別
    (1-1) 類別名稱: BadNeighbor
    (1-2) 建構元: 傳入[編號](no), [待檢查字串](string) 等2個參數.
                  將參數存入實體中, 成為實體的成員.
    (1-3) [計算距離]方法, distance(self).
          找出self.string中, 每個字元與'惡'的最近距離.
          註: 如果是'惡', 它自己的距離是0; 它旁邊的鄰居是1, 接著是2, 依此類推.
              一個字串可能有多個'惡', 至少有一個'惡'.
          
          例如: '我惡我我我惡'
          ==>   '101210'
          
(2) 產生實體
(3) 顯示資料
```

#### 測試
``` python
1, '我我我我我我我我我我我我我我我我我我我惡'
2, '惡我我我我我我我我我我我我我我我我'
3, '我惡我我我惡'
4, '我我我惡我我我我我我惡我我我我我我惡我我我我'
5, '惡我我我我我我我惡我我我我我我我我惡我我我我我我我惡'
```

